# FreeCAD_Macros
A collection of hopefully useful macros

To use: Download the files into your FreeCAD Macro Directory. The directory can be found by clicking on the Macro menu item and then clicking on Macros. The directory will be in the bottom of the dialog box.

ColorList.FCMacro:
This macro opens a dialog with a list of colors. The user can quickly color objects in the FreeCAD document.
The user can make new color lists, and load / save them.

SelectIntersectCut.FCMacro:
This macro lets you select one object and then a set of objects. It creates a clone for each of the selected objects and performs an intersect or cut to see the resulting shape. https://youtu.be/EZzOezN8l8E
